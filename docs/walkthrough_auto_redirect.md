# 実装内容のまとめ：YouTubeチャンネル動画タブ自動リダイレクト機能

YouTubeチャンネルのホーム（@ユーザー名）にアクセスした際、自動的に「動画」タブへ遷移する機能を追加しました。

## 変更点

### 1. コンテンツスクリプト (`src/content.js`)
- `SpeedController` クラスに `handleRedirect()` メソッドを実装しました。
- `MutationObserver` の監視対象に `handleRedirect()` を含め、YouTubeのSPA遷移（ページ読み込みなしの遷移）でもURLの変化を検知してリダイレクトするようにしました。
- 正規表現を用いて、チャンネルホーム（`/@username` または `/@username/featured`）のみを対象とし、`/videos` タブへリダイレクトします。

### 2. ポップアップUI (`index.html`)
- リダイレクト機能の有効/無効を切り替えるトグルスイッチを追加しました。
- `ConfigManager` を使用して、設定値 (`autoVideoTab`) を永続化するロジックを実装しました。

### 3. スタイル (`styles/main.css`)
- YouTubeのデザインに合わせた、モダンなトグルスイッチ（赤色のスライダー）のスタイルを追加しました。

## 使い方
1. 拡張機能のポップアップを開きます。
2. 「チャンネル時に『動画』タブへ」のトグルをオンにします。
3. YouTubeで任意のチャンネル（例: `@youtube`）にアクセスすると、自動的に `/videos` タブへ移動します。

## 注意事項
- この機能は `ConfigManager` を通じて `localStorage` に保存されます。ポップアップで設定を変更した後、YouTubeのページをリロードするか、新しいページに遷移したタイミングで反映されます。
