# 実装計画書: YouTube再生速度コントロールスイッチ追加

YouTubeの動画プレイヤー下部（コントロールバー）に、再生速度を素早く切り替えられるスイッチを追加します。

## 1. 概要
YouTube標準の再生速度変更は、設定アイコン（歯車）をクリックしてメニューから選択する必要があり、手間に感じることがあります。
本機能では、コントロールバーに直接「x1.0」「x1.5」「x2.0」などのクイックボタン、あるいはシームレスなスイッチを配置し、ワンクリックで速度変更を可能にします。

## 2. 実装方針

### 2.1 構成
- **ブラウザ拡張機能 (Chrome Extension)** として実装。
- **Manifest V3** を使用。
- **Content Scripts** でYouTubeのDOMを監視・操作。

### 2.2 技術スタック
- **JavaScript (Vanilla JS)**: 軽量で高速なDOM操作。
- **CSS (Vanilla CSS)**: YouTubeの既存デザインに完璧に馴染むプレミアムなUI。

## 3. 具体的な実装ステップ

### Step 1: プロジェクト基盤の構築
- `manifest.json` の作成（`youtube.com` へのアクセス許可設定）。
- 現状のテンプレートから拡張機能に必要なファイル構造へ整理。

### Step 2: 動画プレイヤーの検知とUI注入
- YouTubeのプレイヤー要素（`#movie_player`）を検知。
- コントロールバー（`.ytp-right-controls` 付近）にカスタムコンテナを注入。
- ページの遷移（SPA遷移）に対応するための `MutationObserver` の実装。

### Step 3: 再生速度制御ロジック
- `video.playbackRate` を直接操作。
- 現在の速度をボタンのスタイル（アクティブ状態）に反映。

### Step 4: UI/UXデザイン
- **プレミアム感**: YouTubeのアイコンや色使いに合わせた透過感のあるボタン。
- **マイクロインタラクション**: ホバー時のエフェクト、切り替え時の滑らかなインジケーター表示。

## 4. デザイン案
- **配置箇所**: 動画プレイヤー右下の設定アイコンの左隣。
- **UI構成**: 
  - `[ 1.0 ]` `[ 1.5 ]` `[ 2.0 ]` というシンプルなテキストボタン。
